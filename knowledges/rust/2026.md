# 四、推理 Runtime 层（LLM Inference）

这是 Rust 的传统强项之一

---

## 1️⃣ 本地 LLM 推理

### 核心方案

```
llama.cpp bindings (Rust)
+ candle
+ ggml / gguf
```

---

### 关键组件

#### Candle

Rust 原生 ML 框架（轻量版 PyTorch）

优势：

* 纯 Rust
* 嵌入式友好
* Apple Silicon 强

适用：

* 边缘推理
* 自定义模型部署

---

#### llama.cpp Rust bindings

用途：

* 调用本地 GGUF 模型
* Ollama 底层很多用这个

---

## 2️⃣ GPU 推理

* CUDA bindings 逐步成熟

未来：

> Rust + WebGPU 推理会爆发
